## 🚀 Быстрый старт (30 секунд)

1. Открой 1. Открой [**генератор промтов**](https://cognify-ui.github.io/ai-prompt-generator)
2. Выбери категорию или воспользуйся поиском
3. Нажми на понравившийся промт → нажми «Скопировать»
4. Вставь промт в [**Cognify AI**](https://cognify-ui.github.io) или любой другой AI-чат
5. Получи идеальный ответ!

> 💡 **Совет:** Промты можно и нужно адаптировать под свою задачу — заменяй текст в квадратных скобках `[например, так]`.

## 💡 Почему это работает?

Качественный промт определяет **90% успеха** при работе с ИИ. Наши шаблоны:

- ✅ Протестированы на реальных задачах
- ✅ Содержат структурные маркеры (роли, контекст, формат ответа)
- ✅ Подходят для любой современной нейросети
- ✅ Экономят часы на экспериментах

**Пример плохого промта:**
> «Напиши пост»

**Пример хорошего промта (наш шаблон):**
> «Напиши продающий пост для Instagram про [продукт]. Используй эмоциональный заголовок, список преимуществ, вопрос к аудитории и призыв к действию. Тон: дружелюбный. Длина: 150-200 слов.»

Разница в результате — колоссальная!

## 🔧 Локальный запуск (для разработчиков)

```bash
# Клонируем репозиторий
git clone https://github.com/cognify-ui/ai-prompt-generator.git
cd ai-prompt-generator

# Открываем в браузере (никаких зависимостей!)
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux

# Или через локальный сервер (рекомендуется)
python -m http.server 8000
# → открываем http://localhost:8000
